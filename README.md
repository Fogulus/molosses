# Molosses

Plateforme centralisée de gestion de pedigrees et élevages canins.

## Stack prévue

- Backend : Django / DRF
- Frontend : HTMX / Alpine / Tailwind
- Database : PostgreSQL + MongoDB
- Infra : Docker

## Structure

- backend/ → API et logique métier
- frontend/ → interface utilisateur
- docs/ → conception, UML, Merise
- infra/ → docker, déploiement

## Objectif

Créer une plateforme scalable pour éleveurs et propriétaires de molosses.
